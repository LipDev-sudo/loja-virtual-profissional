# Loja Virtual Profissional

Template profissional de loja virtual com design moderno e funcionalidades completas de e-commerce.

## Preview

[Ver Demo](https://loja-virtual-profissional.vercel.app/)

## Tecnologias

- React
- TypeScript
- Tailwind CSS
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Catalogo de produtos com filtros
- Carrinho de compras
- Design profissional e responsivo
- Layout otimizado para conversao
- Componentes reutilizaveis
